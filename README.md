## 📖 Описание проекта

Программа предназначена для автоматического резервного копирования изображений из популярных API в облачное хранилище Яндекс.Диск. Она позволяет:

1. **Получать изображения кошек** с настраиваемым текстом через API [cataas.com](https://cataas.com)
2. **Получать изображения собак** различных пород через API [dog.ceo](https://dog.ceo)
3. **Автоматически создавать структуру папок** на Яндекс.Диске
4. **Сохранять детальную информацию** о каждой загрузке в JSON-файл
5. **Обрабатывать ошибки** и конфликты при загрузке

### Основные особенности

- ✅ **Загрузка "по воздуху"** — изображения загружаются напрямую в облако без локального сохранения
- ✅ **Автоматическое создание уникальных имен** — при конфликте имен файлов добавляется временная метка
- ✅ **Индикатор прогресса** — визуальное отображение хода загрузки
- ✅ **Детальное логирование** — все действия записываются в лог-файлы
- ✅ **Поддержка русского языка** — корректная обработка кириллицы в тексте и именах файлов
- ✅ **Обработка подпород** — для собак автоматически загружаются изображения всех подпород

## 🎯 Функциональные возможности

### Для кошек (cataas.com)
- Загрузка изображений с произвольным текстом
- Настройка размера, цвета и шрифта текста
- Предустановки стилей (default, large, funny, elegant, minimal)
- Пакетная загрузка (до 10 изображений за раз)

### Для собак (dog.ceo)
- Загрузка изображений всех пород
- Загрузка конкретной породы
- Автоматическая обработка подпород
- Сохранение в структурированные папки по породам

### Для Яндекс.Диска
- Автоматическое создание папок
- Загрузка файлов без локального сохранения
- Обработка конфликтов имен
- Получение информации о загруженных файлах
